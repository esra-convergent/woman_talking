# Avatar Assets

Place your avatar image layers here.

## Required Files

All files must be PNG format with transparent backgrounds:

- `head.png` - Base head image (without mouth)
- `eyes-open.png` - Open eyes layer
- `eyes-closed.png` - Closed eyes layer
- `mouth-A.png` - Wide open mouth (ah, I)
- `mouth-B.png` - Lips together (m, b, p)
- `mouth-C.png` - Slightly open (e)
- `mouth-D.png` - Tongue/teeth (t, d, s, z)
- `mouth-E.png` - Rounded lips (o)
- `mouth-F.png` - Lower lip/teeth (f, v)
- `mouth-G.png` - Throat/back (k, g)
- `mouth-H.png` - Aspirated (h)
- `mouth-X.png` - Closed/rest

## Image Specifications

- **Format:** PNG with transparency
- **Size:** All images must be the same dimensions
- **Recommended:** 512x512px or 1024x1024px
- **Alignment:** All layers must align perfectly when overlaid
